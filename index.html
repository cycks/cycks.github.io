Languages

I am not sure it is appropriate to say that I have been married to R for the last five years. I have tried to divorce R for Python but I realize that R is a better tool for data science. The main reason being, the R community is full of scholars that appreciate the need of driving the message home when communicating. Simply put, learning to use a new tool in R is easier than learning to use a new tool in python because R documentations are easier to understand and follow. Still, I know python and like it too. Apart from the mentioned languages I can also use javascript, jquery, HTML, and CSS. These, I just learn for fun.
